Для владельца сайта

Как подключить заявки и админку

Формы продуктов, рецептов и статей сохраняют заявки в Supabase. В админке можно проверить материал, исправить поля и нажать «Одобрить» — после этого он появляется на сайте.

Без ручной правки JSON и GitHub
1

Создайте проект Supabase

После создания проекта откройте SQL Editor и запустите файл docs/supabase-schema.sql. Перед запуском замените YOUR_ADMIN_EMAIL на свой email.

2

Создайте пользователя владельца

В Supabase откройте Authentication → Users, добавьте пользователя с тем же email и задайте пароль. Этим email и паролем вы будете входить в owner-panel.html.

3

Вставьте ключи проекта

Откройте docs/js/submission-config.js и вставьте туда Project URL, anon key и свой adminEmail. Secret/service_role ключи на сайт не вставляйте.

4

Проверьте полный путь

Отправьте тестовый продукт через форму, откройте админку, войдите, нажмите «Одобрить» и проверьте, что продукт появился в каталоге.

Как работает безопасность

Посетители могут только отправлять новые заявки и читать уже одобренные материалы. Просматривать ожидающие заявки и менять статус может только пользователь Supabase Auth с email, который вы указали в SQL-политиках.
