Модерация в статической версии

Настоящая Flask-админка не работает на GitHub Pages, потому что там нет Python, сессий и SQLite. В этой версии модерация делается через GitHub:

  1. Пользователь отправляет заявку через форму рецепта или форму статьи.
  2. Ты проверяешь созданный Issue в репозитории.
  3. Если всё подходит, переносишь данные в data/dishes.json или data/myths.json.
  4. Делаешь commit, и GitHub Pages обновляет сайт.

Старая страница admin_moderation.html оставлена как памятка, чтобы структура проекта не терялась.
